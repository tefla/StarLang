## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
A browser-based 3D voxel editor (similar to Kenney Shape) for creating game assets.

Core concept: Draw on a 2D pixel canvas, assign depth values, generate 3D voxel models.

Key features:
- 2D pixel canvas (up to 32x32 or 64x64)
- Per-pixel depth/extrusion control
- Color palette (8-16 colors)
- Real-time 3D preview
- Export to glTF for use in-game

This will be used collaboratively to create assets for StarLang.
